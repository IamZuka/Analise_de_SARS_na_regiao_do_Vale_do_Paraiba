#Analise_SARS_no_Vale_do_Paraiba
Esse projeto visa demonstrar o perfil de pessoas infectadas nas principais cidades do vale do paraiba, para isso, foi utilizado de graficos demonstrando sexo, etnia, localidade, escolaridade e idade dos individuos infectados pelo covid na região.

Os dados utilizados nesse projeto podem ser obtidos no https://opendatasus.saude.gov.br/dataset/bd-srag-2020
